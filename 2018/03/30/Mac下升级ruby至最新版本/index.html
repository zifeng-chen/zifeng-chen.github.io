Mac下升级ruby至最新版本

2018年3月30日

Mac OS自身安装的有ruby，但是版本较低，为2.0.x
而安装CocoaPods，最低版本是2.2

查看ruby版本 

ruby --version

安装homebrew 

ruby -e "$(curl -fsSL https://raw.github.com/Homebrew/homebrew/go/install)"

通过 brew 安装 Ruby 最新版本 

1.  brew update
2.  brew install ruby

再次查看ruby版本 

ruby --version

添加ruby至环境变量 

1.  vim .profile
2.  vim .bashrc
3.  vim .bash_profile

重启source 

1.  source .profile
2.  source .bash_profile
